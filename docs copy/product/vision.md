# Vision Produit

## Contexte

Overbound est une course à obstacles prévue en septembre 2026.
Le produit sert de CRM léger pour piloter les partenariats (prospection, relance, négociation, signature).

## Problèmes à résoudre

- Suivi insuffisant avec Excel seul.
- Relances peu fiables (oubli, manque de priorisation).
- Pas de vue consolidée d'avancement.
- Envoi d'emails et pièces jointes non industrialisé.

## Objectifs V1

- Dashboard de suivi de pipeline partenaires.
- Fiche partenaire avec historique et prochaine action.
- Relances avec rappels in-app et push.
- Génération de brouillons d'emails via IA.
- Envoi d'emails avec plaquette par défaut.

## Hors périmètre V1

- Automations avancées multi-campagnes.
- Scoring IA complexe.
- BI avancée.

## Acteurs

- Utilisateur principal: fondateur (single user au démarrage).
- Cible technique: multi-utilisateur activable sans refonte.
